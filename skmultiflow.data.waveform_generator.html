class skmultiflow.data.waveform_generator.WaveformGenerator(random_state=None, has_noise=False)[source]Â¶

Bases: skmultiflow.data.base_stream.Stream

Generates instances with 21 numeric attributes and 3 classes, based on a random differentiation of some base waveforms. Supports noise addition, but in this case the generator will have 40 attribute instances

Parameters:

  • has_noise (bool) â if True additional 19 insignificant will be added. (Default: False)

next_sample(batch_size=1)[source]Â¶

An instance is generated based on the parameters passed. If noise is included the total number of features will be 40, if itâs not included there will be 21 attributes. In both cases there is one classification task, which chooses one between three labels.

After the number of attributes is chosen, the algorithm will randomly choose one of the hard coded waveforms, as well as random multipliers. For each attribute, the actual value generated will be a a combination of the hard coded functions, with the multipliers and a random value.

Furthermore, if noise is added the features from 21 to 40 will be replaced with a random normal value.

Parameters:batch_size (int) â The number of samples to return.
Returns:Return a tuple with the features matrix and the labels matrix for the batch_size samples that were requested.
Return type:tuple or tuple list
