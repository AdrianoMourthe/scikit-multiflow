skmultiflow.classification.base module¶

class skmultiflow.classification.base.BaseClassifier[source]¶

Bases: skmultiflow.core.base_object.BaseObject

The abstract class that works as a base model for all of this framework’s classifiers. It creates a basic interface that classifiers should follow in order to use them with all the tools available in scikit-workflow.

This class should not me instantiated, as none of its methods, except the get_class_type, are implemented.

Raises:NotImplementedError: This is an abstract class.
