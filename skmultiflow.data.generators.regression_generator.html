skmultiflow.data.generators.regression_generator module¶

class skmultiflow.data.generators.regression_generator.RegressionGenerator(n_samples=40000, n_features=100, n_informative=10, n_targets=1)[source]¶

Bases: skmultiflow.data.base_instance_stream.BaseInstanceStream

This generator creates a stream of samples for a regression problem. It uses the make_regression function from scikit-learn, which creates a batch setting regression problem. These samples are then sequentially fed by the next_instance function.

Parameters:
  • n_samples (int (Default: 40000)) – Total amount of samples to generate.
  • n_features (int (Default: 100)) – Number of features to generate.
  • n_informative (int (Default: 10)) – Number of relevant features, in other words, the number of features that influence the class label.
  • n_targets (int (Default: 1)) – Number of targeting tasks to generate.
