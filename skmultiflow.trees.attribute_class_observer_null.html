skmultiflow.trees.attribute_class_observer_null module¶

class skmultiflow.trees.attribute_class_observer_null.AttributeClassObserverNull[source]¶

Bases: skmultiflow.trees.attribute_class_observer.AttributeClassObserver

Class for observing the class data distribution for a null attribute. This method is used to disable the observation for an attribute. Used in decision trees to monitor data statistics on leaves.
