skmultiflow.neural_networks.perceptron module¶

class skmultiflow.neural_networks.perceptron.PerceptronMask(penalty=None, alpha=0.0001, fit_intercept=True, max_iter=1000, tol=0.001, shuffle=True, verbose=0, eta0=1.0, n_jobs=1, random_state=0, class_weight=None, warm_start=False)[source]¶

Bases: skmultiflow.core.base.StreamModel

A mask for scikit-learn’s Perceptron classifier.

Because scikit-multiflow’s framework require a few interfaces, not present int scikit-learn, this mask allows the first to use classifiers native to the latter.
